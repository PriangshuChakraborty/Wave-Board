# Wave-Board

Welcome to Wave Board, a go-to platform for real-time whiteboard collaboration. With Wave Board, you can brainstorm ideas, teach your team or students etc.

## Features

- **Real-time Collaboration :** Collaborate with multiple users simultaneously on the same whiteboard.
- **Drawing Tools :** Utilize a variety of drawing tools including shapes, lines and freehand sketches with customizable colors.
- **Text :** Add text boxes to annotate the whiteboard.
- **Simple Interface :** Intuitive UI design for easy navigation and usage.
- **Chat :** Communicate with other in real-time using the built-in chat feature.
- **User Presence :** See who has joined/left the session and the all users.
- **Host Identification :** Identify the host of the session with a distinct label.
- **Reactions :** React to the host's presentation with emojis and also you can raise your hand.
- **Save and Export :** Save your sessions and export them in PDF formats.

## Getting Started

To get started with Wave Board, go to the website : [Wave-Board](https://wave-board.vercel.app/)
## Usage

1. If you are hosting a session go to `create room` and after creat a room share the code to others. 
2. If you are not a host go to `join room` and using the code join the room.
3. As a host start collaborating with your team members by drawing, adding text and othes in the session will see it.
4. Chat with other users, react about the presenter's presentation and if you not understand raise your hand. 
5. Save your whiteboard session when you're done, or export it for further use.

## Technologies Used

- **React.js :** Frontend framework for building the user interface.
- **Tailwind CSS :** Utility-first CSS framework for styling the user interface.
- **Socket.io :** Real-time communication between clients for collaborative features.
- **HTML5 Canvas :** Drawing and rendering tools for the whiteboard.
- **Vercel :** Used for deploying the frontend of the application.
- **Render :** Used for deploying the backend server of the application.
